  1. Configuración del cliente SSH en equipos personales para agilizar conexión SSH:
# Abrir una terminal en el equipo cliente y crear un directorio llamado .ssh:

mkdir ~/.ssh

# Cambio de permisos al directorio ~/.ssh
cd
chmod -R go= ~/.ssh

# Crear y editar el archivo ~/.ssh/config
# Contenido archivo config:

Host ometeotl.atmosfera.unam.mx
    Hostname 132.248.8.155
    Port 9022
    User usuario_ometeotl
    ServerAliveInterval 120
    #IdentityFile ~/.ssh/usuario_ometeotl

# Los parámetros anteriores varían, según las características de la
# conexión entre el equipo cliente y el servidor. Guardar los cambios

# Probar la conexión:
ssh ometeotl.atmosfera.unam.mx
